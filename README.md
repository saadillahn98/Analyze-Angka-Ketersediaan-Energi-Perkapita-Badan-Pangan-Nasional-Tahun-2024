# Analyze-Angka-Ketersediaan-Energi-Perkapita-Badan-Pangan-Nasional-Tahun-2024

Exploratory Data Analis (EDA) adalah langkah kunci dalam analisis data, yang berfokus pada pemahaman pola, tren, dan hubungan melalui alat statistik dan visualisasi. Python menawarkan library yang kuat seperti Pandas, NumPy, Matplotlib, Seaborn, dan Plotly, yang memungkinkan eksplorasi dan pembuatan wawasan yang efektif untuk memandu pemodelan dan analisis lebih lanjut.

Dalam artikel ini, kami akan melakukan pra-proses, dan melakukan Analisis menggunakan Bahasa Pemograman Python. Data yang kami analisis yaitu data tabular (structured data) tentang “Angka Ketersedian Energi Perkapita Badan Pangan Nasional Tahun 2024” yang dilansir oleh Satu Data Indonesia: https://data.go.id/dataset/dataset/angka-ketersediaan-energi-per-kapita-update-tahun-2024
